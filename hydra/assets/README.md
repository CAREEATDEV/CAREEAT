# Assets à fournir

- `icon.png` (1024×1024, goutte segmentée) — REQUIRED
- `splash.png` (fond noir, logo centré) — REQUIRED
- `fonts/ChakraPetch-Bold.ttf`
- `fonts/ChakraPetch-SemiBold.ttf`
- `fonts/IBMPlexMono-Regular.ttf`
- `fonts/IBMPlexMono-Bold.ttf`

Sources :
- Chakra Petch : https://fonts.google.com/specimen/Chakra+Petch (OFL)
- IBM Plex Mono : https://fonts.google.com/specimen/IBM+Plex+Mono (OFL)

Placer chaque .ttf ici, `App.tsx` les charge via `expo-font`.
